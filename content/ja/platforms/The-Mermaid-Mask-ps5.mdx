<Section icon={Swords} title="キャラクターと容疑者">

**The Mermaid Mask PS5**の謎を解くには、潜水艦内の階級制度を理解することが不可欠です。マグナス・モルトゥーガの死後、権力の空白が即座に緊張を引き起こします。各キャラクターには、殺人に関連しているかもしれないし、していないかもしれない特定の「汚れた秘密」があり、犯人を見つけるためには個人的なドラマのノイズを振り分ける必要があります。

| キャラクター | 役割 | 潜在的な動機 |
| :--- | :--- | :--- |
| **マグナス・モルトゥーガ** | 前船長 | 被害者（強い意志を持つ男） |
| **ミス・シーフォーム** | 船長代理 | 権力掌握、遺産相続 |
| **学者** | 研究者 | 禁断の知識、呪い |
| **技師** | 技術責任者 | サボタージュ、反乱 |
| **料理人** | 乗組員支援 | モルトゥーガとの隠された過去 |

会話からは、マグナス・モルトゥーガが「強大な力」を持つ男であったことが示唆されており、彼の密室での突然の死をさらに疑わしいものにしています。乗組員の集団的な運命は、「学術的な確信」と通路で囁かれる「暗く恐ろしい」秘密を見分けるあなたの能力にかかっています。

<Section icon={Shield} title="PS5の技術的パフォーマンスと機能">

**The Mermaid Mask**のPlayStation 5版は、コンソールのユニークなハードウェアを最大限に活用するように設計されています。潜水艦の閉所恐怖症的な雰囲気は、3Dオーディオと精密なハプティックフィードバックによって強化され、船体のあらゆるきしみが直接的で脅威的に感じられます。

| 機能 | PS5での強化点 | プレイヤーのメリット |
| :--- | :--- | :--- |
| **解像度** | ネイティブ4K / 60 FPS | 証拠のテクスチャが非常に鮮明 |
| **ロード** | 超高速SSD | セクター間のシームレスな移動 |
| **ハプティクス** | DualSenseの統合 | 潜水艦の振動を体感できる |

| **トリガー** | アダプティブトリガー | 錆びたドアを開ける際の抵抗感 |

視覚的な忠実度は、犯行現場の調査において特に重要です。溜まった血や床で見つかった謎の物体の質感など、細部が高解像度のアセットで描写され、重要な手がかりを提供します。
