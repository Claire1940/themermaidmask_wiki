<Section icon={Sparkles} title="ヘビのキャビネットパズル">

食堂には、キャビネットが組み込まれた大きな金属製のヘビの像があります。このキャビネットは4つの回転ダイヤルでロックされており、それぞれに異なる幾何学図形が描かれています。これは **The Mermaid Mask 体験版攻略**における最初の難関です。

**パズルの論理:**
各図形の「辺の数」が、周囲の環境で見える数字、あるいはダイヤル配列の内部論理に対応している必要があります。

| ダイヤルの位置 | 必要な図形 | 辺の数 |
| :--- | :--- | :--- |
| **左** | 三角形 | 3辺 |
| **上** | 四角形 | 4辺 |
| **右** | 七角形 | 7辺 |
| **下** | 九角形 | 9辺 |

パズルを解くとキャビネットが開き、「太陽の錠剤（Sun Tablets）」のボトルが現れます。この錠剤には3つの種類があり、後に犯行現場で見つかる証拠と比較する際の重要なポイントになります。

<Section icon={Swords} title="イリュージョン・シアターの捜査">

体験版のクライマックスは、イリュージョン・シアターで展開されます。ここがマグナス・モルトゥーガが命を落とした場所です。部屋は狭く、被害者は「密室」状態で発見されたため、捜査は非常に困難を極めます。

**現場で発見された証拠:**
- **血痕**: モルトゥーガの首から部屋の奥に向かって流れ、水溜まりになっています。
- **月の瓶 (Moon Vial)**: 月のアイコンが付いた、植物の香りがする砕けた紫色のガラス瓶。
- **手描きのポスター**: 「星を見つめる少女（The Stargazing Girl）」というタイトルで、元々は逆さまに掲示されていました。
- **大釜**: 内側に深いひっかき傷があり、外側にフジツボが付着した古代の石の容器。

| 証拠アイテム | 場所 | 重要性 |
| :--- | :--- | :--- |
| **砕けた瓶** | 床 | 紫色の液体が入っており、太陽の錠剤とは対照的。 |
| **石の大釜** | 中央 | 長年ロックされていたが、現在は謎の開錠がなされている。 |
| **帆布** | 壁 | 星を見つめる少女の絵が描かれている。 |
| **逆さまのポスター** | パイプ | フォーチュナ・ホールの「鏡の中の貴婦人」について言及。 |

<Callout type="warning" title="重要な推論">
モルトゥーガは武器が見当たらない密室で一人でした。部屋にある唯一の他の物体は石の大釜です。内部の傷を注意深く調べてください。何かが「外に出よう」としていたことを示唆しています。
</Callout>

</Section>

<Section icon={Trophy} title="最終的な推理と体験版の結末">

体験版は、グリモワールがモルトゥーガの死の経緯を繋ぎ合わせる推理シーケンスで締めくくられます。大釜の状態と死因を関連付ける必要があります。

**推理の重要ポイント:**
- モルトゥーガは喉を鋭利に切り裂かれて死亡した。
- 部屋の中に刃物は存在しなかった。
- 大釜は歴史上初めて、最近になって解錠された。
- **結論**: モルトゥーガは大釜の封印を解き、その中にいた「何か」によって殺害された。

<Callout type="tip" title="最後のステップ">
推理の後、体験版は「第1章：大釜の呪い（The Curse in the Cauldron）」というタイトルで終了します。事前にデッキにある「木の少年」の船首像を調べておきましょう。その光る目は、緑色の光に関する環境パズルの最後のピースとなります。
</Callout>
