- 潜水艦「モルトゥガ」の浮上と不気味な緑色の光を目撃する。

<Section icon={Sparkles} title="蛇の像のパズルを解く">

**マーメイド・マスクの攻略**における主要な機械的障壁の一つが、潜水艦内にある巨大な金属製の蛇の像です。この像にはキャビネットが組み込まれており、4つのダイヤルを使った複雑なロック機構があります。

<Steps>
  <Step num="1" title="像の場所を特定する">
    談話室とキッチンエリアの境界付近にある金属製の蛇の像を見つけます。
  </Step>
  <Step num="2" title="ダイヤルを調べる">
    キャビネットを調べて4つのダイヤルを表示させます。各ダイヤルの中央には異なる図形が見える窓があります。
  </Step>
  <Step num="3" title="図形を合わせる">
    「舵取りの呪い」や港の噴水で見つけた手がかりをもとに、ダイヤルを回して図形を合わせます。
  </Step>
  <Step num="4" title="キャビネットを開ける">
    正しいシーケンスを入力すると、キャビネットが開き、マグナス・モルトゥガに関するさらなる手がかりが明らかになります。
  </Step>
</Steps>

このパズルには細部への鋭い観察力が必要です。蛇自体は、町の石の噴水と船の金属像の両方に現れる繰り返しのモチーフです。足のある生き物を果たして「蛇（サーペント）」と呼べるのかという、ヒートとサリーの議論にも注目してください。

| ダイヤルの位置 | 図形／シンボル | 関連性 |
| :--- | :--- | :--- |
| **ダイヤル 1** | 蛇の頭 | 噴水の石像に関連 |
| **ダイヤル 2** | 船／乗り物 | 愛しきセレナ号に関連 |
| **ダイヤル 3** | 波／海 | 海事のテーマに関連 |
| **ダイヤル 4** | 謎のシンボル | 事件ファイルのメモから判断 |

<Callout type="success" title="パズル攻略の戦略">
ダイヤルパズルで行き詰まったら、港の噴水をもう一度確認してください。漁師と格闘する「シルクワーム・サーペント」の描写が、キャビネットを解錠するために必要なシンボルのヒントになっています。
</Callout>

<Section icon={Shield} title="イリュージョン・シアターの事件現場">

体験版のクライマックスは「イリュージョン・シアター」で起こります。ここはマグナス・モルトゥガが遺体で発見された場所です。部屋は非常に狭く、実質的にはドアで仕切られた行き止まりの通路です。この「密室」での殺人が、**マーメイド・マスクの攻略**における最大の謎となります。

**事件現場の観察:**
- **遺体**: マグナス・モルトゥガは仰向けに倒れており、足元には大釜があった。
- **傷口**: 首に鋭い切り傷があるが、密室内に武器は見当たらなかった。
- **大釜**: 事件前には一度も開けられたことがないと報告されている謎の容器。
- **家具**: ストーン・トップ・インのバースツールと同一に見える、倒れたスツール。

| 手がかり | 観察内容 | 潜在的な意味 |
| :--- | :--- | :--- |
| **割れたビン** | 床の上で発見 | 争っている最中にモルトゥガが落としたもの |
| **逆さまのポスター** | 壁に貼られている | 部屋の用途変更（リブランディング）の可能性 |
| **ひっかき傷** | 大釜の内側 | 中に何かが閉じ込められていたことを示唆 |
| **血痕** | 遺体まで続いている | 襲撃地点を示している |

イリュージョン・シアター自体は、標準的な物置部屋を最近になって「リブランディング」したもののようです。壁にかかった布や「イリュージョン・シアター」と描かれたバナーの存在は、殺人が起こる前に、この部屋が特定の目的や公演のために設営されたことを示唆しています。

<Callout type="warning" title="大釜の謎">
大釜は最も不気味な要素です。中身は空に見えますが、内側のひっかき傷は、あんなに小さな容器の中に人が隠れることは物理的に不可能であるにもかかわらず、モルトゥガがそれを開けて「中にいたもの」によって殺された可能性を示唆しています。
</Callout>

</Section>
