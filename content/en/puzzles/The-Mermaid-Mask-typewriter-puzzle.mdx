<Section icon={Swords} title="Step-by-Step Solution to The Mermaid Mask Typewriter Puzzle">

Once you have gathered the Cipher Sheet, the Ink Ribbon, and the clues from the suspect cabins, return to the Captain's quarters. Follow these instructions carefully to input the code and unlock the hidden compartment.

<Steps>
  <Step num="1" title="Prepare the Typewriter">
    Interact with the typewriter on the desk. Open the top cover and insert the **Ink Ribbon** into the dual spools. Use the **Brass Key** to unlock the bottom drawer, which reveals the mechanical gear mechanism.
  </Step>
  <Step num="2" title="Align the Cipher Gears">
    Rotate the gears underneath the typewriter to match the symbols on the **Cipher Sheet**. This aligns the internal tumblers, allowing the keys to strike the paper correctly without jamming.
  </Step>
  <Step num="3" title="Input the Decrypted Code">
    Type the word **ZEPHYR** using the substitution keys. Because the keys M, A, S, and K are substituted, you must press the corresponding keys based on the cipher translation table.
  </Step>
  <Step num="4" title="Retrieve the Evidence">
    Once the final letter is typed, the carriage return lever will slide to the right automatically. A hidden compartment behind the typewriter will pop open, revealing the Captain's Logbook.
  </Step>
</Steps>

To ensure you press the correct keys during Step 3, refer to the substitution table below. Pressing the wrong key sequence will lock the keyboard, requiring you to reset the gears.

| Missing Key | Substitution Letter | Decrypted Output |
| :--- | :--- | :--- |
| **Key M** | Z | First character of the final code |
| **Key A** | E | Second character of the final code |
| **Key S** | P | Third character of the final code |
| **Key K** | H | Fourth character of the final code |

<Callout type="tip" title="Pro Tip">
If the keys lock up, pull the silver lever on the left side of the typewriter twice. This action resets the input mechanism without losing your overall puzzle progress.
</Callout>

</Section>
